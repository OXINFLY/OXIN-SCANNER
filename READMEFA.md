🛡️ اسکنر اکسین (ابزار تشخیص IP اندروید)
اسکنر اکسین (OXIN SCANNER) یک ابزار تشخیصی شبکه اندروید ممتاز، قوی و فوق‌العاده بهینه شده است که برای اسکن، آزمایش و فیلتر کردن آدرس‌های IP پاک طراحی شده است. این ابزار بررسی‌های اعتبارسنجی چندلایه دقیقی را انجام می‌دهد، از فایروال‌ها عبور می‌کند و میان‌افزار را جعل می‌کند تا نقاط ورودی پایدار شبکه را با دقت میلی‌ثانیه مشخص کند.

این برنامه با دقت فراوان بازطراحی شده است تا اسکن‌های فوق‌العاده سریع، عملکرد بدون تأخیر در سخت‌افزارهای ضعیف (مثلاً سامسونگ گلکسی A12) و رابط‌های کاربری مدرن و واکنش‌گرا با تم سایبری را که کاملاً بر پایه Jetpack Compose و Material Design 3 ساخته شده‌اند، ارائه دهد .

🎯 قابلیت‌های کلیدی
⚡ موتور همزمانی : از کوروتین‌های کاتلین، فلو کاتلین و یک ساختار سمافور سفارشی برای اجرای همزمان حداکثر ۴۵ تست IP موازی بدون محدودیت CPU استفاده می‌کند.
🛠️ اعتبارسنجی پیشرفته سه لایه : با تأیید یکپارچگی شبکه در سه لایه (TCP، TLS/SNI، HTTP/1.1)، کلاهبرداری Middlebox، ذخیره‌سازی پروکسی و تغییر مسیرهای مسیریابی را کاملاً خنثی می‌کند.
📱 بهینه‌سازی دستگاه‌های رده پایین : مکانیک‌های رندرینگ سفارشی ساخته شده‌اند تا برنامه حتی روی پردازنده‌های هشت هسته‌ای ارزان‌قیمت (رم ۳ گیگابایت) هم روان بماند.
🌐 کنسول گزارش‌گیری تعاملی و بلادرنگ : بازخوردهای رنگی را برای تشخیص IPهای پاک‌شده، اسکن‌شده و ناموفق به‌صورت پویا منتشر می‌کند.

این برنامه کاملاً با الگوهای معماری پاک تحت الگوی MVVM (Model-View-ViewModel) مطابقت دارد تا نگرانی‌ها را از هم جدا کرده و یکپارچگی چرخه عمر را در طول اسکن‌های پویای شبکه حفظ کند:

لایه وضعیت/خط لوله ( DashboardViewModel.kt) : ورودی‌ها، پیکربندی‌های اعتبارسنجی، جمع‌آوری ناهمزمان کارها و انتشار وضعیت را مدیریت می‌کند. گزارش‌های ورودی را در یک صف امن برای نخ رهگیری می‌کند تا جریان‌های وضعیت رابط کاربری را به‌صورت دسته‌ای به‌روزرسانی کند.
موتور شبکه ( IpTester.kt) : عملیات شبکه بومی در سطح سوکت را مستقیماً روی خطوط لوله نخ Kotlin JVM پیاده‌سازی می‌کند.
📑 خط لوله تأیید تست چند لایه

🔹 لایه ۴: TCP Handshake ( InetSocketAddress)
یک اتصال سوکت خام استاندارد را به سمت IP هدف روی پورت 443(HTTPS) آغاز می‌کند. این مرحله، در دسترس بودن مسیریابی و تأخیر اولیه تبادل بسته TCP را تأیید می‌کند.

🔹 لایه ۶: مذاکره سفارشی TLS/SSL ( SSLSocket)
سوکت متصل خام را با استفاده از ... به صورت ایمن پوشش می‌دهد SSLSocketFactory. این به صورت دستی هدرهای میزبان SNI (نشانگر نام سرور) سفارشی پیکربندی شده توسط کاربر را به بسته TLS client-hello تزریق می‌کند. این باعث می‌شود sslSocket.startHandshake()اعتبارسنجی رمزنگاری تکمیل شود. این سطح، IP های مسدود شده در لایه TLS handshake را شناسایی و حذف می‌کند.

🔹 لایه ۷: تأیید دست‌دهی HTTP Payload ( HTTP HEAD)
یک هدر ساده HTTP سفارشی صریح را از طریق لینک امن ایجاد شده ارسال می‌کند:

HEAD / HTTP/1.1
Host: <your-sni-hostname>
User-Agent: OXIN-Scanner/2.5 (Android)
Accept: */*
Connection: close
خط وضعیت پروتکل HTTP ورودی را مستقیماً از طریق جریان سوکت تجزیه می‌کند. یک IP فقط در صورتی Clean ONLY اعلام می‌شود که با موفقیت یک ساختار خط وضعیت سازگار با HTTP (مثلاً ، 200 OK، 302 Foundو غیره) را برگرداند. این کار نقاط انتهایی مسیریابی ساختگی، دستکاری ISP و پیکربندی‌های رهاسازی بار را ایزوله و از بین می‌برد.

⚡ طرح صافی دستگاه سطح پایین

ادغام همزمانی ایمن نخ : لاگ‌ها به صورت غیرمسدودکننده ثبت می‌شوند ConcurrentLinkedQueue. به جای به‌روزرسانی فوری وضعیت Compose برای هر لاگ (که باعث قفل شدن نخ‌های رابط کاربری می‌شود)، یک کوروتین تخصصی، لاگ‌های پس‌زمینه را هر ۳۵۰ میلی‌ثانیه به صورت دسته‌ای به رابط کاربری ارسال می‌کند .
محدودکننده‌های رندرینگ LazyColumn :
فهرست خام لاگ‌های کنسول بلادرنگ، درون تابع MainActivityusing قرار می‌گیرد takeLast(30)تا لاگ‌های قدیمی از حافظه حذف شوند.
فهرست پیشرفت اسکن فعال با استفاده از محدود می‌شود take(25)تا فقط مرتبط‌ترین جهش‌های بلادرنگ رندر شوند.
کلیدهای پایدار ساختاری صریح : هر دو طرح‌بندی لیست اکنون از کلیدهای حالت صریح ( key = { it.ip }و key = { it }) درون LazyColumns استفاده می‌کنند. این امر مانع از آن می‌شود که Jetpack Compose هنگام تغییر سایر موارد، موارد خارج از صفحه را دوباره ترسیم کند.
به خاطر سپردن وضعیت پایدار : عملیات محاسباتی پرهزینه درون چرخه‌های طرح‌بندی (مانند تعیین isCleanیا isFailedرشته‌های وضعیت) درون بلوک‌های Compose قرار می‌گیرند remember(logMessage)و عملکرد رابط کاربری را فوق‌العاده سریع نگه می‌دارند.
🎨 هویت بصری و آیکون‌های پویا
تم مدرن : از یک رابط کاربری سایبرپانک تاریک فوق‌العاده باکیفیت با پرتقال‌های مسی درخشان با کنتراست بالا و هایلایت‌های فیروزه‌ای نئونی بهره می‌برد.
آیکون لانچر سفارشی : از قبل با یک آیکون وکتور لانچر برنامه با جزئیات بالا بارگذاری شده است که ترکیبی از زیرساخت ابری، حلقه‌های اسکن رادار، یک سپر فلزی محافظ و یک «X» کرومی پررنگ است.
